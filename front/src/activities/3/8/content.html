Un codage fréquent : le code à décalage

Les codes à décalage sont couramment utilisés, il n’est pas de bon jeu de piste qui n’y ait recours ! Ils sont basés sur un principe simple : on décale l’alphabet d’un certain nombre de lettres. Par exemple A devient F, B devient G etc… jusqu’à Z devient E. On peut aussi utiliser des chiffres. A devient 6, B devient 7 etc… jusqu’à Z qui devient 5.

Certains codes sont très célèbres car on peut glisser la clé de codage dans un seul mot (ou un dessin). C’est le cas pour celui que tu connais surement qui est AVOCAT (A vaut K).

Pour te faire découvrir certains nouveaux codes à décalages, voici un mini jeu : ta mission est de relier le nom du code à la relation entre 2 lettres (ou entre une lettre et un chiffre) à laquelle il correspond ! Le code à décalage : Le chiffrement et le déchiffrement mais Quésako ?

Le chiffrement

Le chiffrement correspond au moment où tu codes ton message. Par exemple, si tu utilises A=K et que tu veux que celui qui suis ta piste trouvent la lettre A, tu notes K.

Le déchiffrement

Le déchiffrement, c’est l’inverse, c’est le moment où l’on décode un message. Avec A=K, si le joueur voit K écrit, il doit remonter à la lettre A pour comprendre ton message. Par contre, si le joueur voit la lettre A écrite, il doit remonter à la lettre Q.

Quand tu codes un message, tu dois donc faire attention aux lettres que tu écris. Par exemple pour coder « bonjour », avec le code Avocat, tu notes « lyxtyeb ».

Qu’est-ce que le code à symbole ?

Pour faire un code à symbole, il faut transformer ton texte en une succession de symbole. Tu peux inventer ton propre alphabet, mais, dans ce cas, tu dois donner à la personne qui décode ton alphabet avec les lettres correspondant à chaque symbole. Ou alors tu peux utiliser un code à symbole qui existe déjà comme le morse ou le code des templiers.

Comment les codes peuvent être complexifiés ?

Quand tu connais les bonnes techniques, tu peux décoder n’importe quels codes en 2 minutes chrono. Voici 2 méthodes pour rendre un code plus difficile !

Le surcodage

Le surcodage consiste à coder 2 fois de suite le même message avec 2 codes secrets différents.

Attention à ne pas utiliser deux codes à décalage (Avocat, Cassis…), car ça agrandira (ou réduira) seulement le décalage sans le surcoder. (Si ton premier code A=1, et que ton second code 1=B, alors A=B). Tu peux utiliser 2 codes à décalage uniquement si ton premier code est à lettres et le 2ème à chiffre et lettre.

Voici un exemple, codé en Avocat (A=K), puis cassis (K=6) :
En clair : Ceci est un message
En Avocat : moms ocd ex wocckqo
En cassis : 8.10.8.14..10.24.25..26.19..18.10.24.24.6.12.10.

Si tu le souhaites, tu peux t’entrainer à décoder en décodant d’abord le code codé en cassis puis celui en avocat. Regarde si tu retrouves « ceci est un message ».

Comment les codes peuvent être complexifiés ?

L'écriture à l'envers

L’écriture à l’envers consiste à écrire la phrase à l’envers puis à coder cette phrase écrite à l’envers.

Voici un exemple, mis à l’envers puis codé en Hélène (L=N) :
En clair : Un code plus difficile
A l’envers : eliciffid sulp edoc nu
En Hélène : gnkekhhkf uwnr gfqe pw.

Si tu le souhaites, tu peux t’entrainer à décoder en décodant d’abord le code codé en Hélène puis remettant le texte à l’endroit. Regarde si tu retrouves « un code plus difficile ».
